# Jeet Jani - Portfolio

Welcome to my personal portfolio site built with React, TypeScript, Tailwind CSS, and Vite.

## Features

- Responsive Design
- Contact Form (EmailJS)
- Modern UI (shadcn-ui)
- GitHub integration

## Deployment
